# Dictionary for DerivRMCode variable of RCI2D file

Dictionary for DerivRMCode variable of RCI2D file

## Usage

``` r
RCI2D__DerivRMCode
```

## Format

### `RCI2D__DerivRMCode`

A data frame with 11 rows and 2 columns:

- code:

  Integer code

- value:

  Character description
