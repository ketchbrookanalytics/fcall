# Dictionary for ExposureCode variable of RCI2C file

Dictionary for ExposureCode variable of RCI2C file

## Usage

``` r
RCI2C__ExposureCode
```

## Format

### `RCI2C__ExposureCode`

A data frame with 12 rows and 2 columns:

- code:

  Integer code

- value:

  Character description
