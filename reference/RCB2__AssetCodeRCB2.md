# Dictionary for AssetCodeRCB2 variable of RCB2 file

Dictionary for AssetCodeRCB2 variable of RCB2 file

## Usage

``` r
RCB2__AssetCodeRCB2
```

## Format

### `RCB2__AssetCodeRCB2`

A data frame with 17 rows and 2 columns:

- code:

  Integer code

- value:

  Character description
