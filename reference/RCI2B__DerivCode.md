# Dictionary for DerivCode variable of RCI2B file

Dictionary for DerivCode variable of RCI2B file

## Usage

``` r
RCI2B__DerivCode
```

## Format

### `RCI2B__DerivCode`

A data frame with 23 rows and 2 columns:

- code:

  Integer code

- value:

  Character description
